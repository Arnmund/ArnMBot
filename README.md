# ArnMBot
 
Arn M Bot — программа, которая следит за чатом вашего
Twitch-канала и по команде от модератора автоматически
перезагружает указанную вкладку в браузере. Удобно для
обновления оверлеев без ручного нажатия F5.

Бот подключается к Twitch анонимно, без токена, и может
одновременно слушать до 5 разных каналов — каждый в отдельной
вкладке интерфейса.


----------------------------------------------------------------
2. УСТАНОВКА — РАСПАКОВКА АРХИВА
----------------------------------------------------------------

Бот поставляется в ZIP-архиве. Перед запуском его необходимо
распаковать.

  ! ВАЖНО: Не запускайте бот прямо из ZIP-архива —
    он не сохранит настройки и может работать некорректно.

Как распаковать:

  1. Найдите скачанный файл ArnMBot.zip в папке Загрузки.
  2. Нажмите на него правой кнопкой мыши.
  3. Выберите «Извлечь всё…» или «Распаковать в текущую папку».
  4. Дождитесь завершения распаковки.
  5. Откройте появившуюся папку ArnMBot.

Содержимое папки после распаковки:

  ArnMBot.exe  — исполняемый файл бота
  icon.png     — иконка программы
  README.txt    — это руководство

Папку можно разместить в любом удобном месте: Рабочий стол,
Документы, диск D: и т.д. Главное — не удалять файлы из неё.


----------------------------------------------------------------
3. ПОДГОТОВКА БРАУЗЕРА
----------------------------------------------------------------

Бот управляет браузером через протокол CDP (Chrome DevTools
Protocol). Для этого браузер нужно запустить с особым флагом,
открывающим порт для отладки.

Поддерживаются все Chromium-браузеры:
Google Chrome, Vivaldi, Microsoft Edge, Opera, Brave.


3.1 Создание ярлыка с флагом отладки
--------------------------------------

  1. Найдите ярлык браузера на рабочем столе или в меню Пуск.
  2. Нажмите правой кнопкой мыши → «Свойства».
  3. В поле «Объект» найдите путь к браузеру.
  4. Добавьте в конец строки (после кавычек) нужные флаги
     (см. ниже для каждого браузера).
  5. Нажмите «Применить» и «ОК».
  6. Теперь всегда запускайте браузер только через этот ярлык.

  ! ВАЖНО: Если запустить браузер обычным способом (без флага),
    бот не сможет к нему подключиться.


3.2 Флаги для каждого браузера
--------------------------------

  Vivaldi:
    --remote-debugging-port=9222 --user-data-dir="C:\VivaldiDebug"

  Google Chrome:
    --remote-debugging-port=9222 --user-data-dir="C:\ChromeDebug"

  Microsoft Edge:
    --remote-debugging-port=9222 --user-data-dir="C:\EdgeDebug"

  Opera / Brave:
    --remote-debugging-port=9222 --user-data-dir="C:\BrowserDebug"

Пример итоговой строки для Vivaldi:

  "C:\Program Files\Vivaldi\Application\vivaldi.exe"
  --remote-debugging-port=9222 --user-data-dir="C:\VivaldiDebug"

  * Примечание: --user-data-dir создаёт отдельный профиль
    специально для работы с ботом. Ваши основные закладки
    и данные не затрагиваются.


3.3 Открытие оверлея
----------------------

  1. Запустите браузер через созданный ярлык.
  2. Откройте в нём вкладку с нужным оверлеем или страницей.
  3. Скопируйте URL из адресной строки — он понадобится
     при настройке бота.


----------------------------------------------------------------
4. ЗАПУСК И НАСТРОЙКА БОТА
----------------------------------------------------------------

4.1 Первый запуск
-------------------

  1. Откройте папку ArnMBot.
  2. Дважды щёлкните на ArnMBot.exe.
  3. Если Windows покажет предупреждение «Защитник Windows» —
     нажмите «Подробнее» → «Выполнить в любом случае».

  * Это стандартное предупреждение для программ без цифровой
    подписи. Бот безопасен.


4.2 Интерфейс
---------------

Окно бота содержит вкладки вверху — по одной на каждый канал.
Переключаться между ними можно кликом по кнопкам
«Вкладка 1» … «Вкладка 5».

Символ и цвет на кнопке вкладки показывает статус бота:

  ○  серый    — Офлайн (бот не запущен)
  ◌  жёлтый  — Подключение...
  ●  зелёный — Онлайн (бот слушает чат)
  ✕  красный — Ошибка подключения


4.3 Настройки — секция TWITCH
--------------------------------

  Канал     — имя стримера на Twitch (без @ и без пробелов).
              

  Префикс   — символ перед командой. По умолчанию: !

  Команда   — слово команды без префикса. По умолчанию: reload
               Итоговая команда в чате: !reload

  Только для модераторов — если галочка установлена, команду
               могут использовать только модераторы и стример.
               Рекомендуется оставить включённым.


4.4 Настройки — секция БРАУЗЕР (CDP)
---------------------------------------

  CDP адрес  — адрес порта отладки браузера.
               По умолчанию: http://localhost:9222
               Менять не нужно, если использовали порт 9222.

  URL оверлея — полный адрес вкладки, которую нужно
               перезагружать. Например:
               https://overlay.example.com/scene1


4.5 Проверка подключения к браузеру (Тест CDP)
-------------------------------------------------

Перед запуском бота убедитесь, что браузер запущен и оверлей
открыт. Нажмите кнопку «Тест CDP».

В логе появится одно из сообщений:

  CDP доступен. Вкладок: N       — браузер найден, всё готово
  
  Вкладка найдена: [название]    — оверлей найден
  
  CDP недоступен                 — браузер не запущен или
                                   запущен без флага отладки
                                   
  Вкладка с URL '...' не найдена — проверьте URL оверлея


4.6 Запуск и остановка
------------------------

  - Нажмите ЗАПУСТИТЬ — статус изменится на «Онлайн».
  - Нажмите СТОП для остановки.

  * Настройки сохраняются автоматически при запуске бота
    и восстанавливаются при следующем открытии программы.


----------------------------------------------------------------
5. РАБОТА С НЕСКОЛЬКИМИ КАНАЛАМИ
----------------------------------------------------------------

Бот поддерживает до 5 независимых конфигураций. Каждая вкладка
(Вкладка 1 … Вкладка 5) — отдельный бот со своими настройками.

  1. Перейдите на нужную вкладку (например, Вкладка 2).
  2. Заполните поля: другой канал, другой URL оверлея.
  3. Нажмите ЗАПУСТИТЬ на этой вкладке.

Каждая вкладка запускается и останавливается независимо.
Все 5 ботов могут работать одновременно.

Пример:
  Вкладка 1 — команда !reload, канал stream1, оверлей №1
  Вкладка 2 — команда !refresh, канал stream2, оверлей №2


----------------------------------------------------------------
6. ЧАСТЫЕ ПРОБЛЕМЫ И РЕШЕНИЯ
----------------------------------------------------------------

Бот не подключается к чату
  - Проверьте, что название канала введено без @ и пробелов.
  - Убедитесь, что есть подключение к интернету.
  - Остановите и запустите бота снова.

CDP недоступен
  - Убедитесь, что браузер запущен через ярлык с флагом
    --remote-debugging-port=9222.
  - Не должно быть других окон браузера без этого флага.
  - Проверьте, что порт 9222 не заблокирован антивирусом.

Вкладка с оверлеем не найдена
  - Скопируйте URL прямо из адресной строки браузера.
  - URL должен совпадать полностью, включая https://.
  - Убедитесь, что вкладка с оверлеем открыта прямо сейчас.

Команда не срабатывает
  - Проверьте написание команды в чате (регистр не важен).
  - Если включена опция «Только для модераторов» — команду
    может отправить только мод или стример.
  - Убедитесь, что бот онлайн (● зелёный статус).

Ошибка при запуске .exe
  - Убедитесь, что папка распакована из ZIP.
    Запуск прямо из архива не поддерживается.
  - Попробуйте запустить от имени администратора:
    ПКМ на ArnMBot.exe → «Запуск от имени администратора».
